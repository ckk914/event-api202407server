$ sudo curl -L "https://github.com/docker/compose/releases/download/v2.11.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
​
실행 권한 부여
$ sudo chmod +x /usr/local/bin/docker-compose
